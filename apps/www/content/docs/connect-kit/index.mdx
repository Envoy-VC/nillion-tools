Connect Kit is a [React](https://reactjs.org/) library that provides a modal for connecting to the Interchain. It is built on top of [Graz](https://graz.sh/) so you can use any graz hooks to interact with the Interchain.

## Features

- 💳 Out-of-the-box wallet management with connection, balance fetch, wallet activity and more.
- 🪝 Support for Mordern Frameworks such as Next.js, React, Vite, Svelte and more.
- 🧩 Built on top of `graz` for maximum compatibility.
- 🎨 Highly Customizable interface to match your branding.
- 🤝🤝 Supports wide rannge of wallets such as Keplr, Leap, Vectis, Cosmostation, and more.
- 👏 Fully typed and tree-shakeable
- ...and many more ✨

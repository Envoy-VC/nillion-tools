## User Hook

The `useUser` hook is used to retrieve the user information. It returns the user object with the following properties:

- **`activeChains`**: List of active chains connected to the wallet.
- **`chainInfo`**: Information about the Active chain.
- **`currency`**: Currency of the active chain.
- **`account`**: Account information.
- **`balance`**: Balance information.
  - **`formatted`**: Formatted balance.
  - **`isLoading`**: Loading state.

```tsx
import { useUser } from '@nillion-tools/connect-kit';

const Page = () => {
  const { activeChains, balance } = useUser();

  return (
    <div>
      <h1>User Information</h1>
      <p>Active Chains: {activeChains.map((c) => c.chainName).join(', ')}</p>
      <p>Balance: {balance.formatted}</p>
    </div>
  );
};
```

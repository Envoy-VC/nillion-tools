## Modal Options

This helps you customize the modal options like modal size, title and footer.

```tsx
import { ConnectButton } from '@nillion-tools/connect-kit';

const Page = () => {
  return (
    <ConnectButton
      modalOptions={{
        size: 'wide',
        title: 'Sign In',
        privacyPolicyUrl: 'https://example.com/privacy-policy',
        termsOfServiceUrl: 'https://example.com/terms-of-service',
        welcomeScreen: <div>Welcome Screen</div>,
      }}
    />
  );
};
```
